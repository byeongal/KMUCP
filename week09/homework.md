## 1번 문제 ( 2점 )
아래의 코드를 활용하여 두 자연수 `n`과 `k`를 입력받아 `1`부터 `n`까지의 자연수 중 `k`의 배수의 합을 구하는 코드를 작성하세요.
### 입력 예시
```
100 7
```
### 출력 예시
```
735
```

### 참고 코드
```python
n, k = map(int, input().split())
# n, k 확인용 코드
# print("n :", n, "k :", k)
total = 0
'''
코드
'''
print(total)
```
